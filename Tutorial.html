Advanced Network Merge

Cytoscape allows for merging of both network and table data


Merge Networks

  • Select the required number of networks to merge from the list



  • The Preview will look like the below



  • Select one of the Three Operations



  • Enter the Merged network name


  • Select whether to follow in network merge



  • Select whether to Ignore Edges



  • The Final Preview is shown below



  • Choose the node keys and edge keys. For the network,node,edge column matchers you can add more lists by selecting in cycles. one Cycle is defined as the fillup of options starting from the first network and ending at the last "Merged Column Datatype".

  • Click on merge button and See the results in your Local instance Cytoscape


Layouts


  • From "Get Available Layouts" heading click on "get layouts" to get a list of available Layouts and choose a Layout

  • To get the Details of the selected layout click on "Get Description" button

  • To apply the selected layout on the network click on "Apply" button


Visual Styles


  • Click on "get List" to get a list of available Styles and choose a style from the list

  • To get the Details of Current style click on "Get Descripton" button

  • To apply the selected style on the network click on "Apply" button
